## Cách chạy dự án với Node.js

Có nhiều cách để chạy ứng dụng này:

### Phương pháp 1: Dùng server Node.js tích hợp (Đề xuất)

```bash
# Cài đặt dependencies (chỉ cần chạy lần đầu)
npm install

# Khởi động server
npm start
```

Server sẽ chạy tại: http://localhost:3000

### Phương pháp 2: Dùng live-server (tự động reload)

```bash
# Cài đặt live-server (nếu chưa có)
npm install -g live-server

# Hoặc dùng script có sẵn
npm run dev
```

### Phương pháp 3: Dùng http-server

```bash
# Dùng script có sẵn
npm run serve
```
